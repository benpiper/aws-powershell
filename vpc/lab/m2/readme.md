# VPC Lab — Module 2

## Scripts

- [vpc-creation.ps1](vpc-creation.ps1) — Create VPCs, subnets, route tables, security groups, and instances.
- [instancedrilldown.ps1](instancedrilldown.ps1) — Given the Name tag of an instance, display the primary ENI and its associated security groups, subnet, route table, and routes.
